# 二、改进1

## MaskDINO UAV Pavement Crack Opt-v1

### 1. 实验目的

本实验针对 UAV Pavement Crack Dataset 上的裂缝实例分割任务，对 MaskDINO-R50 基线模型进行稳妥优化。原始模型在验证集上虽然具有较高的 AP50，但 AP75 和细长裂缝类别 AP 较低，说明模型能够较好地定位裂缝区域，但 mask 边界精度和细长裂缝分割质量仍有提升空间。

本轮优化重点关注以下目标：

- 提升实例分割主指标 segm AP；
- 提升严格 IoU 条件下的 segm AP75；
- 改善 transverse_crack、longitudinal_crack、oblique_crack 三类细长裂缝的 AP；
- 尽量保持较高 AP50，不破坏模型基本检测能力。

---

### 2. 数据集与类别

数据集：UAV Pavement Crack Dataset  
任务类型：实例分割 / 目标检测  
模型：MaskDINO-R50  
验证集：pavement_cracks_uav_val

类别共 4 类：

| 类别               | 含义     |
| ------------------ | -------- |
| transverse_crack   | 横向裂缝 |
| longitudinal_crack | 纵向裂缝 |
| alligator_crack    | 龟裂     |
| oblique_crack      | 斜向裂缝 |

---
